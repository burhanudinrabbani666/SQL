## Data Definition vs Manipulation

### Data Definition

- Data definition are query statements, so SQL commands so to say, that define databases, tables, and table structures
- Table contraints & relationship
- Databsase & table management(e.g altering tables, droping tables)

### Ddata Manipulation

- Query statement that manipulation(insert, update, delete) ot fetch data.
- May Join, Filter or sort data
- Use database & tables that were cretaed via "Data definition statment"
