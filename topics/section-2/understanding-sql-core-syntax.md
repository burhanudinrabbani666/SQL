## Understanding SQL core Syntax

```sql
SELECT name, salary FROM employees WHERE salary > 8000;
```

- whenever write SQL statement, should be end with a semicolon.
- The token is basically each individual part our SQL statement consists of.

- Keywords (Token): Keywords in SQL define the actual action that should be performed based on our command. Exm: SELECT, FROM, WHERE
- Idenifiers (Token): An identifier is a token that forms a name. An identifier in an SQL statement is an SQL identifier, a parameter marker, or a native identifier. Exm: name, salary, employees, salary.
- operator: Exm: + - / \* etc
- Value / Expression
